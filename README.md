# David Nawee Portfolio

A clean, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main page with hero, about, skills, experience, and contact sections.
- `styles.css` — responsive styling and layout.
- `script.js` — mobile menu toggle and contact form behavior.

## Usage

1. Open `index.html` in a web browser.- Click the resume button to download `resume.pdf` or replace it with your own resume file.2. Customize the content and sections to match your experiences and skills.
3. Deploy to GitHub Pages, Vercel, Netlify, or any static site host.

## Customize

- Update hero text and contact details in `index.html`.
- Add more skills or experience items in the `skills` and `experience` sections.
- Replace colors and typography in `styles.css` to match your branding.
